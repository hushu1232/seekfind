## 四、实际部署测试步骤

### 4.1 前置条件

```
✅ Docker Desktop 已安装
✅ Ollama 已安装
✅ Chrome 浏览器
✅ Node.js 18+ (构建扩展)
```

### 4.2 后端启动测试

```bash
# Step 1: 启动 Ollama
ollama serve
ollama pull qwen2.5:7b
ollama pull nomic-embed-text

# Step 2: 启动后端（Docker）
cd D:/求问/求问
cp .env.example .env
docker compose up -d

# Step 3: 验证
curl http://localhost:8700/health
# 预期: {"status":"ok","model_strategy":"hybrid"}

curl http://localhost:8700/api/status
# 预期: {"ollama":true,"chroma":true,"tts":true,...}
```

### 4.3 扩展构建测试

```bash
# Step 1: 生成图标（需要设计稿或占位图）
# 将 icon16.png, icon48.png, icon128.png 放入 extension/icons/

# Step 2: 安装依赖
cd extension
npm install

# Step 3: 构建
npm run build
# 输出到 extension/dist/

# Step 4: 加载到 Chrome
# chrome://extensions → 开发者模式 → 加载已解压 → extension/dist/
```
